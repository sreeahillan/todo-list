📝 To-Do List App

A simple, responsive To-Do List web application built using HTML, CSS, and JavaScript. This project helped me practice DOM manipulation, event handling, and working with the browser's local storage.

🚀 Features


➕ Add new tasks
❌ Delete tasks
✅ Mark tasks as complete
💾 Tasks are saved in local storage (so your list stays even after refreshing or closing the browser)


🛠️ Built With


HTML – structure of the app
CSS – styling and layout
JavaScript – functionality and logic (DOM manipulation, event listeners, local storage)

⚙️ How to Run Locally


Clone this repository


bash   git clone https://github.com/sreeahillan/todo-list-app.git


Navigate into the project folder


bash   cd todo-list-app


Open index.html in your browser (or use the Live Server extension in VS Code)
